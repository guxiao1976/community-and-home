# OpenSpec 提案模板

## 快速创建提案

```bash
# 1. 创建提案目录
mkdir -p openspec/changes/your-proposal-name
cd openspec/changes/your-proposal-name

# 2. 复制模板文件
cp ../../TEMPLATE.md proposal.md
```

---

## .openspec.yaml

```yaml
schema: spec-driven
created: YYYY-MM-DD
status: draft          # draft | in-progress | completed | cancelled
assignee: username     # 可选
priority: medium       # high | medium | low
estimated_days: 3      # 可选：预估工作天数
```

## API设计

### RESTful API

#### 创建资源
```
POST /api/v1/resource
Content-Type: application/json

Request:
{
  "name": "string",
  "value": "string"
}

Response:
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "name": "string",
    "value": "string"
  }
}
```

### gRPC API

```protobuf
service ResourceService {
  rpc CreateResource(CreateResourceReq) returns (ResourceResp);
}

message CreateResourceReq {
  string name = 1;
  string value = 2;
}

message ResourceResp {
  int64 id = 1;
  string name = 2;
  string value = 3;
}
```
